# Blog thumbnail illustrations

Reference image library for Enzuzo's blog post thumbnail illustration style. Used as visual reference for designers, contractors, and AI image-generation tools (Midjourney, DALL-E, Flux Kontext) when producing new thumbnails.

## What's in this repo

A curated set of reference PNGs exported from the master Figma file, representing the established visual style across Enzuzo's blog post thumbnails. Each image is 1920×1000 px.

## Style guide

The full written style guide — colour palette, typography rules, illustration style, composition principles, and headline emphasis conventions — lives in Notion:

**[Blog thumbnail illustrations — Notion style guide](https://www.notion.so/enzuzo/Blog-thumbnail-illustrations-350148fc9b5280ac8375dac0717c61d8)**

Always read the Notion guide first. This repo only holds the reference images.
